Free Label Designer for Thermal Printer

2024-12-06

UI

It is mainly divided into 6 parts, menu bar, element list area, element attribute editing area, preview area, batch file area, and printer area.

Firstly Choose Printer and Set Paper Size

Select the printer to which you want to print from the printer list.

Set the size of the printing paper in millimeters


Create Label Model

Add elements (Text/Shape/Image/TimeStamp)

modify properties of element

We can ajust position/rotate/scale of all elements
TextElement: FontFamily/Line Space/From File and so on

Some Element Example

I created a Bread Label Model.Element 1/2/4/6/7/8 were fixed text.We can just type the text.
Element 3/5/9 are from file which we can choose file(csv).

Choose File(csv)

Connect File Column to Element

After choosing file we can connect file data to elements.We can choose the column we wanna.

Label Model Management

Open /New /Save Model

In menubar we can manage model with openning / newing / saving.

Printing

Print Options


One is for printing page currently previewed.The other one is printing all pages which are in relation with csv file we choosed.
