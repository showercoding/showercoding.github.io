How to create a clothing tag with free label designer

2024-12-10 | Hits

Software description

According to your own needs, you can print cargo labels, waybill labels, clothing labels, flight labels, etc. There is no charge for this software. I wrote it myself using Python. The download link is at the end of the article, and there is a way to get the activation code for free in the download zip file.

Select printer and set paper

Select the corresponding printer. For any printer on the market, if the driver is installed normally, you can use this software to design and print labels.

Set the paper size in millimeters

The paper can be set in one row and multiple columns (applicable to small labels, some small labels have 2/3/4 sheets in a row)

#Create template
Add template elements (text, pictures, shapes, time and date)

Elements can be adjusted in position, size, angle, etc. According to your own table, numbering and other elements are added here.


If the element content comes from an Excel file, you need to import the Excel file in advance. Select the file first, find the file and confirm the import. Click the element and check From file. Find the corresponding column and the element will be associated with the file. Here, the imported file data is converted into a barcode or a QR code.

Template management

Save the created template in the template management at the top for subsequent use.

Print

There are two options for printing labels. You can print the currently previewed page or print all. The number here is related to the rows count of imported file.
